Privacy Policy

Last updated: February 11, 2026

Brew Scripts is an open-source project by CodeCraftedApps ("we," "our," or "us"). This Privacy Policy explains how data is handled when you use the Brew Scripts toolkit, visit this website, or interact with our GitHub repository.

Our Privacy Commitment

Brew Scripts is designed with a privacy-first approach. As open-source command-line tools that run entirely on your local machine, our scripts are built to respect your privacy by design.

Data the Scripts Collect

Local Data Only

Brew Scripts operates entirely on your local machine. The scripts:

  • Do not transmit any data to external servers owned by us
  • Do not collect analytics, telemetry, or usage statistics
  • Do not phone home or check for updates automatically
  • Do not require account creation or authentication with our services

Log Files

The scripts create log files stored locally on your machine (typically in ~/Library/Logs/). These logs contain:

  • Timestamps of script execution
  • Names of packages installed or updated
  • Success or failure status of operations
  • System information (macOS version, architecture)

These log files remain on your device and are never transmitted to us. You may delete them at any time.

Configuration Files

Configuration files (such as config/homebrew-scripts.conf) are stored locally and may contain settings you provide, such as email addresses or phone numbers for notifications. This data is used only by the scripts running on your machine and is never sent to us.

Third-Party Services

The scripts interact with the following third-party services as part of their core functionality:

  • Homebrew (brew.sh): The scripts download and manage packages through Homebrew, which is subject to Homebrew's own policies
  • GitHub: Source code is hosted on GitHub. Downloading or cloning the repository is subject to GitHub's Privacy Statement
  • Apple iMessage/Mail: If you configure notifications, the scripts use macOS built-in Mail and iMessage apps, which are subject to Apple's Privacy Policy

Website Data

This Website

This website (brewscripts.codecraftedapps.com) is a static site hosted on GitHub Pages. We do not use cookies, tracking pixels, or analytics services on this site.

  • GitHub Pages may collect standard server logs (IP address, browser type, pages visited) as part of hosting. See GitHub's Privacy Statement for details
  • Google Fonts are loaded from Google's CDN, which is subject to Google's Privacy Policy
  • The scripts library loads manifest data from jsDelivr CDN, which is subject to jsDelivr's Privacy Policy

Open-Source Transparency

As an open-source project, all code is publicly available for review on GitHub. You can inspect exactly what the scripts do before running them. This transparency is a core part of our privacy commitment.

Children's Privacy

Brew Scripts is a developer tool and is not directed at children under the age of 13. We do not knowingly collect any personal information from children.

Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated date. As this is an open-source project, significant changes will also be reflected in the repository's commit history.

Parent Organization

Brew Scripts is a project of CodeCraftedApps. For the general privacy policy covering all CodeCraftedApps projects, please see the CodeCraftedApps Privacy Policy. Where this tool-specific policy provides more detail, it takes precedence for matters related to Brew Scripts.

Contact

If you have questions about this Privacy Policy, please reach out through the CodeCraftedApps Contact page or open an issue on GitHub.
